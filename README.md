# Portfolio — Óscar Parra

Web personal de Óscar Parra, realizador y director audiovisual.

## Archivos

- `index.html` — página principal
- `style.css` — estilos

Sitio estático: no necesita compilación ni dependencias.

## Ver en local

Abrir `index.html` en el navegador, o servirlo con:

```bash
python3 -m http.server 8000
```

Y visitar http://localhost:8000

## Despliegue

Conectado a Netlify. Cada `git push` a `main` publica automáticamente.
